# InstaMoney | ATM Machine GUI

## Description

InstaMoney is a user-friendly, cross-platform graphical application built using Python and the Tkinter library. It is a Mordern GUI of Atm Machines.


## Technologies Used

- Python 3.7+
- Tkinter
